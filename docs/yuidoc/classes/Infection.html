Infection Class

Defined in: develop\infect.js:115
Module: vAnalyze

Object returned from calling infect on a Host. Serves as the base for most other vAnalyze functionality.

Example creation:

var myObj = {x:5, y:[10, 12]};
var infection = myObj.infect();

Because infect can be called safely on already infected objects, this type of storage will in practice be largely unecessary.
